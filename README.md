Ini adalah **Single Page Application (SPA)** sekaligus **Progressive Web App (PWA)** untuk **CV Amelia Trans Mandiri**, perusahaan transportasi dan logistik yang beroperasi di seluruh Indonesia.
Aplikasi ini dirancang dengan pendekatan **mobile-first**, tampilan corporate modern, elegan, dan sepenuhnya dibangun dengan **Vanilla JavaScript (ES6+)** tanpa framework berat seperti React/Vue. Tujuannya adalah menyajikan profil perusahaan, biodata tim, dan metrik operasional secara rapi, cepat, dan dapat di-install seperti aplikasi native di Android, iOS, maupun desktop.

Aplikasi ini siap di-host di **GitLab Pages** (atau GitHub Pages) dengan HTTPS gratis, sehingga fitur PWA (service worker, install prompt, offline) berfungsi sempurna.

## Fitur Utama

- **SPA Navigation** tanpa reload halaman (menggunakan hash routing)
- **Sub-menu dinamis** dengan animasi slide/fade
- **Hero section** dengan logo dan slogan animasi fade-in
- **Konten inline** (profil, layanan, armada, biodata, metrik) menggunakan card layout responsif
- **Efek visual modern**:
  - Glow dinamis mengikuti pointer
  - Glassmorphism + soft shadow
  - Hover lift & ripple effect
  - Gradient text & background elegan
- **Fully responsive** & touch-friendly (mobile-first)
- **PWA Features**:
  - Installable (Add to Home Screen)
  - Offline capable (via service worker)
  - Fast loading & caching
- **Ringan**: hanya beberapa file statis, tidak ada dependency eksternal
